安裝依賴庫：

安裝所需的 Python 庫：

`pip install tkinter`

`pip install tkcalendar`

`pip install SQLAlchemy`

`pip install mysql-connector-python`

設定資料庫：

更新 db_config.txt 變數為你的 MySQL 資料庫連接資訊：

`username:your_username`

`password:your_password`

`host:localhost`

`database:memos_db`
